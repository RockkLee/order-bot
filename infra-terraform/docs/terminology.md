# Terminology

- ARN 
  - Amazon Resource Name: An ARN is a unique name of an AWS resource.
- IAM (mainly for services (e.g., ECS, S3, and LB))
  - Define and enforce permissions within an AWS account (especially for roles/services).
- IAM Identity Center (formerly AWS SSO) (mainly for account access management)
  - Manage human login and account access centrally, then hands users a role session into accounts.
